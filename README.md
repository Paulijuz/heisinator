# Elevatorinator

##
To compile the Elevatorinator:
```bash
make
```

To run Elevatorinator:
```bash
./elevator
```

## Doxygen
To generate the Doxygen documentation, run the following command in the root directory of the project:
```bash
make docs
```

## Tests
To compile the unit tests:
```bash
make test
```

To run the tests:
```bash
./unit_tests
```
